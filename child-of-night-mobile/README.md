# 🌙 Child of the Night - Mobile Browser

A secure, privacy-focused mobile web browser with a modern, touch-optimized interface.

## Features

### 📱 Mobile-Optimized Interface
- Touch-friendly buttons and controls
- Bottom navigation for easy thumb reach
- Slide-out menu and panels
- Full safe area support for notches and home indicators
- Status bar with time and battery

### 🔒 Security First
- HTTPS enforcement
- Tracking prevention
- Cookie blocking
- Content Security Policy (CSP)
- URL validation

### 📑 Tab Management
- Multi-tab support
- Quick tab switching
- Tab indicators
- One-tap new tab

### 📖 History & Bookmarks
- Full browsing history
- Quick search history
- Bookmark management
- One-tap bookmarking

### ⚙️ Settings & Privacy
- HTTPS-only mode
- Tracking prevention
- Cookie blocking
- Clear history and cache
- One-tap settings access

## Usage

Open `src/index.html` in a mobile browser or wrap it with React Native/Cordova for native apps.

## Controls

- **Menu Button (☰)**: Access menu and options
- **Address Bar**: Enter URLs or search
- **Bottom Navigation**: Back, Forward, Refresh, Home, Share
- **Tab Bar**: Switch between tabs
- **Plus Button**: Create new tab

## Mobile Features

- Full viewport optimization
- Touch-optimized spacing (minimum 44px tap targets)
- No zoom on double-tap
- Safe area support
- Share API integration
- Status bar styling

## Installation

Simply open `src/index.html` on a mobile device or integrate into your mobile app framework.

## Security

Child of the Night Mobile implements enterprise-grade security:
- Input sanitization
- HTTPS enforcement
- CSP headers
- XSS protection
- Tracking prevention

## Browser Support

- iOS Safari 12+
- Chrome Android 80+
- Firefox Android 68+
- Samsung Internet 10+
